## 🚀 사용자가 해야 할 것

### 지금 바로 할 수 있는 것 (1~2분)

```powershell
# 1. 환경 확인
.\verify-build-setup.ps1

# 2. 빌드 테스트
dotnet build Readier.sln
```

✅ **이것들은 지금 바로 작동합니다!**

### Galaxy A32 배포를 위해 해야 할 것 (5~20분, 1회)

```powershell
# Android Studio 실행
& 'C:\Program Files\Android\Android Studio\bin\studio64.exe'
```

- Android Studio가 열릴 때 설치 마법사가 나타남
- **"Next"를 계속 클릭** → SDK가 자동 다운로드됨
- 다운로드 완료 후 바로 Galaxy A32에 배포 가능
